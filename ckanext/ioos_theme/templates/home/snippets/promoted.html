The IOOS Catalog (https://data.ioos.us/) is an open data portal. This implementation is intended for collecting and managing data held by Romanian public authorities and research institutions and it is realized in the frame of the project SPERO – Space technologies used in the management of disasters and major crises, manifested at local, national and regional levels. SPERO received funding from the Minister of Research and Innovation, UEFISCDI under grant agreement no 3Sol/2017, Programme PNCDI III, duration: April 2017 – March 2020. The beneficiary public authority of this project is “Romanian Space Agency” (ROSA).

The Catalog inventories all the compliant data access service endpoints provided by the interested Organizations in a single metadata repository, for discovery by end users.

The Catalog is populated by ISO 19115 metadata records using DMAC-recommended standard vocabularies and data formats wherever possible (netCDF-CF [1], ACDD [2]). Organizations publish their metadata to web accessible folders, or OGC CS-W services, and the Catalog harvests metadata from these locations on a daily basis.

The Catalog provides a searchable user graphical user interface (https://data.ioos.us/) for interactive data discovery by users, as well as a native API (https://data.ioos.us/api/3) and OGC CS-W (https://data.ioos.us/csw?service=CSW&request=GetCapabilities) -compatible service for machine-based access to its inventory of data products
